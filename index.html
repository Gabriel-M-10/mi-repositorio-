Historia de HTML y sus versiones

HTML

HTML es el lenguaje de marcado estándar utilizado para crear páginas web. Fue creado por Tim Berners-Lee en 1990.

HTML 2.0

HTML 2.0 fue lanzado en 1995 y agregó nuevas características, como formularios y tablas.

HTML 3.2

HTML 3.2 fue lanzado en 1997 y trajo consigo mejoras en la estructura y estilo de las páginas web.

HTML 4.01

HTML 4.01 fue lanzado en 1999 e introdujo elementos semánticos y de presentación, así como soporte para hojas de estilo en cascada (CSS).

XHTML

XHTML (Extensible HTML) fue una versión de HTML que seguía las reglas de XML. Fue introducido en 2000 y combinaba la sintaxis de HTML con la rigidez de XML.

HTML5

HTML5 es la última versión de HTML, introducida en 2014. Incluye nuevas características como etiquetas semánticas, reproducción de audio y video, y capacidades de almacenamiento local.
